🌿 Expense Tracker (React)

A clean, pastel-themed expense tracker built using React.
This project allows users to manage expenses, categorize spending, and view real-time summaries in a simple and user-friendly interface.

---

✨ Features

- ➕ Add new expenses
- 🗑 Delete expenses
- 📊 Real-time total spending summary
- 🧮 Derived category totals
- 🔍 Filter expenses by category
- ⚠️ Spending warning when total exceeds threshold
- 🎨 Soft pastel green UI theme
- ✅ Form validation
- 🧩 Reusable React components

---

🖼 Preview

The application provides:

- Expense input form
- Categorized expense list
- Summary dashboard
- Category spending breakdown

---

🛠 Built With

- React (Functional Components + Hooks)
- JavaScript (ES6+)
- CSS (Custom pastel theme)

🚀 Getting Started

1️⃣ Clone the Repository

git clone https://github.com/YOUR-USERNAME/expense-tracker.git

---

2️⃣ Navigate Into Project

cd expense-tracker

---

3️⃣ Install Dependencies

npm install

---

4️⃣ Run Development Server

npm run dev

or (if using Create React App)

npm start
